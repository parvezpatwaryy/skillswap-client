# SkillSwap — Freelance Micro-Task Platform

SkillSwap is a modern freelance marketplace where clients can post small tasks, and freelancers can submit proposals to get hired. It is a reliable platform designed for fast and efficient project completion.

## 🛠️ Technologies Used
* **Frontend**: Next.js, Tailwind CSS, and Shadcn UI
* **Backend**: Node.js and Express.js
* **Authentication**: BetterAuth (Credential and Google OAuth)
* **Database**: MongoDB (Mongoose)
* **Payment**: Stripe API
* **Deployment**: Vercel

## 🚀 Key Tasks Completed
* **Role-Based Authentication**: Implemented distinct access control for Clients, Freelancers, and Admins.
* **Stripe Payment Integration**: Added a secure payment gateway that updates task status automatically upon successful payment.
* **Dynamic Dashboards**: Built interactive dashboards for Clients, Freelancers, and Admins to manage tasks, proposals, and transactions.
* **Search & Filtering**: Developed functionality to search for tasks by keywords and filter by categories.
* **Server-Side Pagination**: Implemented pagination to load exactly 9 tasks per page from the server.
* **Responsive Design**: Created a fully responsive layout optimized for mobile, tablet, and desktop screens.
* **Security Implementation**: Ensured route protection using JWT and HTTPOnly cookies.
